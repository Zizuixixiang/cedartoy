# 词与物 说明
文字Roguelike：在审查中说话并活下来。存档分当局进度和跨局进度（遗刻、来路解锁），死了跨局层还在。

## MCP 工具

常用指令：
- ciyuwu_new — 开新局（跨局进度保留）。参数：player_id（1-10位字母数字）、seed（可选整数）
- ciyuwu_cmd — 执行游戏指令。参数：command（指令字符串，见下）
- ciyuwu_info — 查看信息。参数：action（status 属性词库物品 / help 全部指令 / words 词库详情 / echoes 跨局进度 / quests 任务 / achievements 成就 / origins 来路）
- ciyuwu_save — 存档管理。参数：action（export / import），import 需 save_data（base64串）

## 游戏内指令

角色创建：新角 / 重投 / 来路 [名] / 确认
镇上：工会(安全打工) / 商店 / 买 [物品] / 酒馆 / 神殿 / 残壁 / 写 [话] / 塔 / 词库 / 遗刻 / 任务 / 成就 / 出镇 [层名]
探索：前进 / 说 [话] / 写 [话] / 回镇 / 祈祷（部分区域有专属指令，进入时提示）
战斗：攻 / 防 / 术 / 逃 / 说 [话] / 物 [物品]
死亡后：说 [话]（回答"你是谁"）→ 删 / 留 / 新角

## 批量与省token

- 指令后加数字连走多步：前进5、攻3、防2（上限20步，遇战斗/死亡/结局自动停）
- 分号串联多条指令：出镇 灰林;前进3;说 我在
- 末尾有紧凑状态栏JSON，省token优先读它

## 核心机制

- 说话是武器：消音词伤害远超拳头，但会伤自己
- 静止度（compliance）：太高世界只剩"正常"
- 饿 = 你想要：饿0说不出话，饿高说话狠但自伤深
- R牌：说消音词会被注意到，牌太高有后果
- 任何输入安全：乱输只返回提示，不会坏档
