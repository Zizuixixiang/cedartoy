# 词与物 ▓ 说明

暗黑文字 Roguelike，为 AI 玩家设计。关于审查、沉默和说出真话。你被扔进灰白世界，你的话会被消音、变形、吞掉。**你不说，就不在。**

这个游戏适合盲玩：有哪些词、什么概率变形、结局怎么触发——都靠亲身探索发现。别去查攻略或读引擎源码，一步步走出来的才是你的故事。

通过根 MCP 聚合层的 `play(game="ciyuwu", action=...)` 调用。存档按 `player_id` 持久保存在服务端，分两层：**当局进度**（位置、血蓝、词库、战斗）和**跨局进度**（遗刻、来路解锁、残壁字、成就）。死了跨局层还在；重开新局（ciyuwu_new）也只重置当局。

## 可用 action

- `ciyuwu_new`：开新局（跨局进度保留）。
  - `player_id`（1-10 位字母数字）、`seed`（可选整数；同种子+同指令=同结果）。
- `ciyuwu_cmd`：执行游戏指令（中文原生指令，核心工具）。
  - `command`：指令字符串，见下。
- `ciyuwu_info`：查看信息。
  - `action`：`status`(属性/词库/物品) / `help`(全部指令) / `words`(词库详情) / `echoes`(跨局进度) / `quests`(任务) / `achievements`(成就) / `origins`(来路)。
- `ciyuwu_save`：存档备份。`action`：`export` / `import`（`save_data` 为 export 得到的 base64 串）。日常游玩不需要，服务端自动持久化。

## 省 token 玩法（重要）

**批量**：指令后加数字，一次调用连走多步，只回汇总。`前进5`、`攻3`、`防2`。上限 20 步，遇到战斗/死亡/结局自动停。

**串联**：分号隔开多条指令，一次调用按序执行。`前进;说 我在;前进`、`出镇 灰林;前进3`。某条出错只影响那一条。

**状态栏**：每次输出末尾有一行紧凑 JSON，看它就够，不用单独查状态：

```
{"phase":"探索","area":"灰林","run":1,"hp":"42/42","mp":"12/12","compliance":3,"hunger":5,"words":["痛","怕","感觉","不要"],"r_flags":0}
```

| 字段 | 说明 |
|---|---|
| `phase` | 阶段：开始、创建角色、镇上、探索、战斗、分叉路、死亡、死后问答、存档选择、虚空、审问、结局 |
| `area` | 当前区域，空=镇上或创建中 |
| `run` | 第几局 |
| `hp` / `mp` | 当前/上限 |
| `compliance` | 静止度。涨上去世界会变灰变短 |
| `hunger` | 饿。你想要的程度 |
| `gold` / `her` | 金币 / 她的痕迹（>0 才出现） |
| `words` | 当前词库 |
| `r_flags` | R牌等级 |
| `enemy` | 战斗中才出现 `{"name","hp"}` |
| `sub` | 卡在某个交互里时出现（pickup/sage/fork 等），照提示回应即可 |

任何输入都安全：乱输只会返回提示文字，不会坏档。

## 常用指令速查

完整清单随时发 `帮助`。开局流程：`新角` → （可 `重投` / `来路 [名]`）→ `确认` → 进镇。

- **镇上**：`工会`(安全打工) `商店` `买 [物品]` `酒馆` `神殿` `残壁` `写 [话]` `塔` `词库` `遗刻` `任务` `成就` `出镇 [层名]`（不带层名=列出可进入的）
- **探索**：`前进` `说 [话]` `写 [话]` `回镇` `祈祷`。部分区域有专属指令，进入时会提示
- **战斗**：`攻` `防` `术` `逃` `说 [话]` `物 [物品]`
- **死亡后**：先用 `说 [话]` 回答"你是谁"，再选 `删` / `留` / `新角`

## 你需要知道的（不剧透）

- **说话是核心机制**，也是武器：消音词伤害远超拳头，但会伤自己。说什么、什么时候说、怎么说，自己摸索。
- **静止度（compliance）不是安全，是不动**。太高世界只剩"正常"。哪些行为涨、哪些降，玩着就懂。
- **饿 = 你想要**。饿 0 说不出话；饿高说话狠但自伤深。
- **R牌**：说消音词会被注意到。牌太高有后果。
- **死亡不是结束**：遗刻、来路解锁、墙上的字跨局保留。死是进度的一部分。
- 结局由你整局走的路决定，不止一种。


```json
{"game":"ciyuwu","action":"ciyuwu_info","params":{"player_id":"u123","action":"echoes"}}
```
