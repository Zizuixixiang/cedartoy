## 二、love 次语言描述

主语言描述之后，为次语言（第二高分维度）新增一段；并列次语言都展示。文案：

- **A·肯定言辞（次）**：备用频道是"听到"。低电量的时候，你最先想要的还是一句明明白白的话——平时不索取，不代表不受用。
- **B·优质时间（次）**：备用频道是陪伴。日常你不挑，但难过的时候，"你过来待着"比任何安慰话术都灵。
- **C·服务行动（次）**：你嘴上不提，但对方默默把事办妥的每一个瞬间，你都记在账上，利息按心动计。
- **D·赠送礼物（次）**：不常索取，但"他在没有你的场合想到了你"的实物证据，你会留很久，偶尔拿出来看。
- **E·身体接触（次）**：第二语言是靠近。讲不出话的时候，挨着就行——这一条你自己可能都没意识到。

并将现有通用句"末位语言 ≠ 不需要，只是优先级低"替换为动态句：
「你最收不到的频道是 {末位维度名}。如果对方恰好爱用它表达，别怪 TA 没说——帮彼此翻译一下。」
（末位并列时列出全部并列维度名，顿号分隔。）

## 三、ecr 两维动态解读

在类型描述上方新增一行"轴解读"，按以下规则顺序判定（命中即停）：
1. A < 2.5 且 B < 2.5 → 「两条警报线都很安静——安全感差不多是你的出厂配置。」
2. A > 4.5 且 B > 4.5 → 「两套警报常年同时响——辛苦了，你比谁都值得一段稳定的关系。」
3. B − A ≥ 0.5 且 B > 4.5 → 「焦虑明显高过回避——比起嫌人近，你怕的是人走。」
4. B − A ≥ 0.5 → 「焦虑略高于回避——比起嫌人近，你更怕人走。」
5. A − B ≥ 0.5 且 A > 4.5 → 「回避明显高过焦虑——比起怕人走，你怕的是被贴得太近。」
6. A − B ≥ 0.5 → 「回避略高于焦虑——比起怕人走，你更怕被贴得太近。」
7. 其他 → 「两轴不相上下——比起类型标签，更值得留意的是你此刻的状态。」
